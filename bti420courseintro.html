Controller

A C# class, in a source code file

Manages one or more views

VS template may add code - our HelloWorld example includes three methods, Index, About, and Contact

These methods respond to HTTP GET requests (from browsers)

A method returns a result that is often rendered in a View

Model

Your app's data, and operations on the data

Data is modelled as C# classes

We treat data as though it's always in-memory; C# has rich language-integrated query (LINQ) functionality
